Projetos em Ciência de Dados

Criação De Um Painel Gerencial Para Negócio Com Streamlit.

Nesse projeto foi utilizado os conceitos de programação em Python, manipulação (ETL), pensamento estratégico e analítico, lógica de negócios, junto com ferramentas de desenvolvimento web como Streamlit e Github, foram utilizadas para construção de um painel gerencial com as principais métricas de negócio de uma empresa de marketplace de delivery de comida.

O resultado desse projeto foi um painel interativo postado em um ambiente Cloud que pode ser acessado através de um link por qualquer dispositivo com acesso a internet.
